🧠 개념 학습 (Foundational Concepts)

#개념핵심 흐름
① Knowledge Distillation — Hinton 2015 → EfficientAD Soft target · Dark knowledge · Feature distillation · Uninformed Students · STFPM · Reverse Distillation · Hard Feature Loss · 9년 타임라인
② Cross-Entropy vs KL Divergence — KD에서 왜 서로 다른 손실? $H(p,q) = D_{KL} + H(p)$ · Hard label의 $H = 0$ · Soft label에서 gradient 동일 · $T^2$ 배수 · PyTorch 인터페이스

#주제핵심
① PatchCore 백본을 대상 제품으로 추가 학습 — 도움이 될까? 방식별 4갈래 비교 · feature collapse 원리 · CFA/CutPaste/ReContrast/Dinomaly 보고치 · 의사결정 트리
② CFA/CutPaste/ReContrast/Dinomaly/AnomalyDINO — PatchCore와 같은 방식인가? AD 패러다임 4분류 · 모델별 메커니즘 비교 · PatchCore 거리 (동일/인접/다름) · P1 재구성 제안
③ EfficientAD에서 PDN은 Teacher와 Student에 동일? PDN 공통 아키텍처 + 출력 채널 차이(384 vs 768) · Autoencoder는 별도 구조 · 학습 방식·초기화 차이
④ EfficientAD의 Teacher 추가 학습은 부적절한가? PatchCore 케이스보다 위험 ↑ · Compound collapse 수식 · 5갈래 적응 경로 · 안전한 대안 (Student/AE 강화, Teacher 교체, 재distillation)

딥러닝 기반 Anomaly Detection 기술을 연구 단계에서 실제 제조 생산 현장에서 활용 가능한 제품화 수준으로 끌어올린다.

🧪 채택안 (3인 연구팀 합의)

ID안핵심 메커니즘
P1 IF-PROTO recall@FPR={0.1%, 0.5%, 1%} + bootstrap CI + per-defect-size + Pareto frontier 표준화
P2 CS-BRIDGE MuSc + AnomalyCLIP zero-shot agreement → pseudo-label → PatchCore 학습 (cold-start 14일)
Phase 4 핵심 발견 (사전 시뮬레이션) MVTec 단일 카테고리 규모(N≤500)에서 recall@FPR=0.1%의 bootstrap CI half-width가 모두 5%p를 초과. SOTA 간 비교가 측정 노이즈에 묻힘 → 본 연구의 비판적 기여 후보.
